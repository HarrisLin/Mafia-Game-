## Code Process

Please commit all code on a separate branch and use a pull request to merge code in.

># update master
>
>git checkout master
>
>git pull origin master
>
># Create a new branch
>
>git branch newbranch
>
>git checkout newbranch
>
># Push the new branch
>
>git push origin newbranch
